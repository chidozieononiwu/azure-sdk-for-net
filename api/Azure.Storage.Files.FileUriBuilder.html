    Class FileUriBuilder

    The FileUriBuilder class provides a convenient way to modify the contents of a Uri instance to point to different Azure Storage resources like an account, share, or file.

    For more information, see https://docs.microsoft.com/en-us/rest/api/storageservices/naming-and-referencing-shares--directories--files--and-metadata.
