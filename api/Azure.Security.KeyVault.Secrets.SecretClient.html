    Class SecretClient

    The SecretClient provides synchronous and asynchronous methods to manage Secret in the Azure Key Vault. The client supports creating, retrieving, updating, deleting, purging, backing up, restoring and listing Secret. The client also supports listing DeletedSecret for a soft-delete enabled Azure Key Vault.
