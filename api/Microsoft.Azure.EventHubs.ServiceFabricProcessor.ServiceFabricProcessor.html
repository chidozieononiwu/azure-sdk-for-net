    ServiceFabricProcessor(Uri, Guid, IReliableStateManager, IStatefulServicePartition, IEventProcessor, String, String, EventProcessorOptions, ICheckpointMananger)

    Constructor. Arguments break down into three groups: (1) Service Fabric objects so this library can access Service Fabric facilities, (2) Event Hub-related arguments which indicate what event hub to receive from and how to process the events, and (3) advanced, which right now consists only of the ability to replace the default reliable dictionary-based checkpoint manager with a user-provided implementation.
