    DeserializeEventGridEvents(Stream)

    Deserializes the provided stream using a default JSON serializer that supports all system event types. A webhook/function that is consuming events can call this function to deserialize EventGrid events. For system events, the Data property of each event in the returned array will be set to the appropriate type (e.g. StorageBlobCreatedEventData). For events on custom topics where the type of the Data property can be of any type, the calling function will have to first add a custom event mapping before calling this function.
