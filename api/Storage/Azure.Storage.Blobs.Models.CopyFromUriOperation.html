    Class CopyFromUriOperation

    An Operation<T> for tracking the status of a StartCopyFromUriAsync(Uri, IDictionary<String, String>, Nullable<BlobAccessConditions>, Nullable<BlobAccessConditions>, CancellationToken) request. Its Value upon succesful completion will be the number of bytes copied.
