    public int BackendPort { get; }
    Property Value
    Int32

    Remarks

    This must be unique within a Batch pool. Acceptable values are between 1 and 65535 except for 22, 3389, 29876 and 29877 as these are reserved.

    FrontendPortRangeEnd

    Gets the last port number in the range of external ports that will be used to provide inbound access to the backendPort on individual compute nodes.

    Declaration
    public int FrontendPortRangeEnd { get; }
    Property Value
    Int32

    Remarks

    Acceptable values range between 1 and 65534 except ports from 50000 to 55000 which are reserved by the Batch service. All ranges within a pool must be distinct and cannot overlap.

    Name

    Gets the name of the endpoint.

    Declaration
    public string Name { get; }
    Property Value
    String

    Remarks

    The name must be unique within a Batch pool, can contain letters, numbers, underscores, periods, and hyphens. Names must start with a letter or number, must end with a letter, number, or underscore, and cannot exceed 77 characters.

    NetworkSecurityGroupRules

    Gets a list of network security group rules that will be applied to the endpoint.

    Declaration
    public IReadOnlyList<NetworkSecurityGroupRule> NetworkSecurityGroupRules { get; }
    Property Value
    IReadOnlyList<NetworkSecurityGroupRule>

    Remarks

    The maximum number of rules that can be specified across all the endpoints on a pool is 25. If no network security group rules are specified, a default rule will be created to allow inbound access to the specified backendPort.
