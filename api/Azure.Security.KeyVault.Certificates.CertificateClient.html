    Class CertificateClient

    The CertificateClient provides synchronous and asynchronous methods to manage Certificates in Azure Key Vault. The client supports creating, retrieving, updating, deleting, purging, backing up, restoring and listing the Certificate, along with managing certificate Issuers and Contacts. The client also supports listing DeletedCertificate for a soft-delete enabled key vault.
